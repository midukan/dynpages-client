Na DynPages, a privacidade e segurança dos dados de nossos usuários são prioridades. Este Aviso de Privacidade descreve como coletamos, usamos, compartilhamos e protegemos suas informações pessoais em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.853/2019) e outras legislações aplicáveis.

Sobre Nosso Aviso

Este Aviso se aplica a todos os serviços e produtos oferecidos pela DynPages. Ele define as diretrizes para assegurar a privacidade e a segurança das informações tratadas no âmbito de nossas plataformas, aplicativos e soluções.

Ao utilizar os serviços da DynPages, você declara estar ciente e de acordo com este Aviso de Privacidade, autorizando o tratamento de seus dados pessoais conforme descrito.

Termos e Definições

  • Dados Pessoais: Qualquer informação que identifique um indivíduo, direta ou indiretamente.
  • Aplicativos e Plataformas DynPages: Aplicativos móveis e interfaces web desenvolvidos para disponibilização e contratação das soluções DynPages.
  • Soluções DynPages: Produtos e serviços oferecidos, incluindo gestão financeira, emissão de notas fiscais, integração, pagamentos e recebimentos, sistema ERP, entre outros.
  • Titular de Dados: Pessoa física a quem os dados pessoais se referem.
  • Tratamento: Qualquer operação realizada com dados pessoais, como coleta, utilização, armazenamento e eliminação.

Seus Direitos

Os titulares de dados têm os seguintes direitos, conforme a legislação:

  • Direito de acesso: Solicitar acesso aos dados pessoais tratados.
  • Direito de retificação: Solicitar a correção de dados incorretos ou incompletos.
  • Direito de revogação do consentimento: Revogar o consentimento para ações de marketing, cookies, etc.
  • Direito de remoção: Solicitar a eliminação de dados pessoais.
  • Direito de restrição: Solicitar a interrupção do processamento de dados pessoais.
  • Direito de oposição: Opor-se ao processamento de dados pessoais.
  • Direito à portabilidade de dados: Solicitar uma cópia dos dados pessoais em formato eletrônico.

Coleta de Dados Pessoais

Coletamos dados pessoais de diversas formas, como:

  • Contato ativo via telefone ou e-mail
  • Acesso ao site e uso de aplicativos e plataformas
  • Cookies e formulários preenchidos

Finalidades da Coleta

  • Contratação de Serviços: Dados como nome, CPF, e-mail, telefone, endereço e informações de pagamento são coletados para a contratação de serviços ou produtos.
  • Marketing: Dados como nome, CPF, telefone, endereço e e-mail são utilizados para ações de marketing e otimização do uso do site.

Compartilhamento de Dados

Compartilhamos dados pessoais com prestadores de serviços, processadores de pagamentos e parceiros publicitários para viabilizar a prestação de nossos serviços e ações de marketing. Também podemos compartilhar dados com autoridades para cumprir obrigações legais.

Dados de Terceiros

Os usuários são responsáveis por obter e fornecer dados de terceiros para execução dos serviços, garantindo a devida autorização legal. Esses dados serão armazenados por um prazo máximo de 90 dias após o encerramento do contrato ou inatividade.

Retenção dos Dados

Armazenamos dados pessoais pelo tempo necessário para cumprir as finalidades do tratamento, respeitando as disposições legais e regulatórias. Dados desnecessários, excessivos ou tratados em desconformidade podem ser excluídos a pedido do titular.

Transferência Internacional de Dados

Podemos transferir e armazenar dados em outros países devido ao uso de serviços em nuvem. As transferências internacionais serão realizadas conforme a legislação aplicável.

Segurança dos Dados

Adotamos medidas rigorosas para proteger os dados pessoais, incluindo testes e técnicas de melhoria, sistemas de segurança renomados, relatórios semanais de identificação de falhas e políticas internas rígidas de acesso a informações.

Alterações no Aviso de Privacidade

Podemos alterar este Aviso para melhorar a proteção dos dados pessoais. Os usuários serão notificados sobre alterações significativas através dos canais registrados. Pequenos ajustes podem ser feitos sem notificação prévia.

Resolução de Conflitos

Este Aviso de Privacidade é regido pelas leis brasileiras. Em caso de conflito, fica eleito o foro da Cidade de Bauru - SP.
